## Setup
### Prerequisites
- Ruby (version 2.7+)
- Bundler
- Jekyll

### Installation
1. Clone the repo: `git clone https://github.com/mateusz-piotrowski/mateusz-piotrowski.github.io.git`
2. Install dependencies: `bundle install`

## Usage
- **Run locally:** `./tools/run.sh` (starts Jekyll server with live reload).
- **Build for production:** `./tools/test.sh` (builds the site to `_site/`).
- Visit `http://localhost:4000` to preview.
